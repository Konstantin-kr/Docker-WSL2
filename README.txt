# Docker-WSL2

### Что это?
Локальный сервер для работы с веб сайтами. Работает на основании WSL2 и Docker контейнеров.

### Как работать с проектом?
1. Скопировать проект к себе в дистрибутив Linux, который у вас в WSL
2. Проверить корректность путей в файлах docker-compose.yml
3. Собрать необходимые компоненты проекта или все. Сборка производится из директории компонента командой docker-compose up -d --build

###  Как добавить сайт?
1. В директорию www добавить свой сайт. Для примера в директории www уже создан сайт my-first-docker-site.ru.
2. Добавим конфигурацию ngnix для добавляемого сайта в директорию nginx/conf.d. Примеры конфигураций ngnix расположены в этой же директории.
3. Добавьте ваш сайт в файл hosts в Windows. Расположен он тут C:\Windows\System32\drivers\etc\hosts.
Что бы узнать IP адрес у вашего дистрибутива в WSL, используйте команду
wsl -d "Ubuntu-20.04" hostname –I
Ubuntu-20.04 – название вашего дистрибутива
4. Перезапустите ngnix и php контейнеры
